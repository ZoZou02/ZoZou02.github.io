## 1. 今日任务看板
### 1.1. 今天之前就到期任务
```tasks
not done
due before {{date}}
```

### 1.2. 今天需要干的
```tasks
not done
due on {{date}}
```

### 1.3. 未来两周
```tasks
not done
due after {{date}}
due before {{date+14d}}
```

### 1.4. 没有设置日期的
```tasks
not done
no due date
```

### 1.5. 今天已完成的
```tasks
done on {{date}}
```
